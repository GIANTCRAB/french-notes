Subject pronouns are used to indicate who is performing the action in a sentence. They replace the subject noun and are essential for conjugating verbs correctly.

je, tu, il, elle, nous, vous, ils/elles

# Singulier


| 1ère personne | 2ière personne | 3ière personne |
| ------------- | -------------- | -------------- |
| Je / J'       | Tu             | Il             |
|               |                | Elle           |
|               |                | On             |

# Pluriel


| 1ère personne | 2ière personne | 3ière personne |
| ------------- | -------------- | -------------- |
| nous          | vous           | Ils            |
|               |                | elles          |
