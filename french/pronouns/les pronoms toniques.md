Stressed pronouns are used to emphasize a person or to replace a noun after a preposition. They are also used in short answers or to express a comparison.

moi, toi, lui/elle, nous, vous , eux/elles

# Singulier


| 1ère personne | 2ière personne | 3ière personne |
| ------------- | -------------- | -------------- |
| moi           | toi            | lui            |
|               |                | elle           |
|               |                | nous           |
