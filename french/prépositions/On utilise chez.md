# domicile

On utilise chez pour "la maison de".

## Par exemple

- Je vais chez nous
- Je vais chez ma grand-mère ce week-end

# Avec une personne (oeuvres)

On utilise chez (le), chez (la), chez (l'), chez (les)

Pour les oeuvres.

## Par exemple

Pour acheter le poisson, je vais chez le poissonnier
